## Rectangle `type`

##### Namespace

Exercice_5

##### Summary

Cette classe est utilisé pour la création et modification de rectangles.

```
    Rectangle rectangle = new Rectangle(1, 2);
    rectangle.SetRectangleHeight(1);
    rectangle.SetRectangleLength(2);
    rectangle.GetPerimeter();
```
